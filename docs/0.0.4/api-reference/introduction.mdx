## Authentication

All API endpoints are authenticated using Authetication Header.

```json
Authorization: <api-key>
```

<Info>
For your initial API call, Authorization header can be set to "".
Store the provided API key securely; it will be required to pass the generated and a valid API Key in Authorization header for subsequent API calls.
</Info>
