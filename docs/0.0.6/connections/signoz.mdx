Doku can automatically send LLM usage data from Doku to SigNoz using the [HTTP Logs API](https://signoz.io/docs/userguide/send-logs-http/)

Make the following curl request to Doku to add a connection for SigNoz. See more on the [Connections API](/0.0.6/api-reference/endpoint/connections/create)

```shell 
curl --request POST \
  --url https://<YOUR_DOKU_URL>/api/connections \
  --header 'Authorization: <DOKU_API_KEY>' \
  --header 'Content-Type: application/json' \
  --data '{
  "Platform": "signoz",
  "LogsUrl": "<YOUR_SIGNOZ_LOGS_URL>",
  "ApiKey": "<YOUR_SIGNOZ_API_KEY>"
}'
```
